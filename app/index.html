BCRA OSINT Demo

Consulta automatica via API publica BCRA Wrapper, evidencia local y registro OSINT.

Nueva consulta

Ingresa CUIT/CUIL/CDI de 11 digitos o DNI de 7/8 digitos. Si ingresas DNI, el sistema calcula CUIT/CUIL posibles y consulta automaticamente.

Modo manual con navegador

Solo para contingencia cuando quieras abrir el portal web de BCRA. La consulta principal usa API y cache.

1
Ingresar CUIT/CUIL/CDI o DNI y presionar Enter.
2
El backend consulta Deudas, Deudas Historicas y Cheques Rechazados.
3
El sistema calcula senal preliminar, confianza y caveats.
4
Se guardan JSON crudo, reporte OSINT y linea historica.
